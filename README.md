# TraideMarket Landing

**The marketplace where trades meet opportunity.**

AI-powered hiring for skilled trades. Pre-vetted workers. WOTC credits captured. Placements made in 24 hours.

## Tech Stack

- Next.js 14
- Tailwind CSS
- Supabase (forms + database)
- Vercel (hosting)

## Development

```bash
npm install
npm run dev
```

## Deployment

Connected to Vercel. Deploys automatically on push to main.
